# **App Name**: Guardian Angel

## Core Features:

- SOS Button: A prominent button for immediate help requests.
- Location Detection: Automatically detect the user's latitude and longitude.
- Message Composition Tool: Generative AI tool to automatically compose a message with location, a Google Maps link, and the user's emergency information, based on a prompt.
- Contact Selection: Select and save loved ones contacts from the user's contact list.
- Emergency Message Templates: Provide pre-defined emergency message templates.
- SMS Sending: Send the emergency message via SMS to selected contacts.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) for trust and security.
- Background color: Light Grayish-Blue (#E8EAF6) for a calm and reliable feel.
- Accent color: Bright Red (#F44336) for the SOS button to signify urgency.
- Body and headline font: 'PT Sans' for readability and a modern feel.
- Use simple, clear icons to represent different emergency situations.
- Prioritize the SOS button for quick access, place it at the center-bottom of the screen for easy reach.
- Subtle animation on the SOS button when pressed to provide feedback to the user.
